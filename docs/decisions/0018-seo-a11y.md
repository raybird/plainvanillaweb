# ADR 0018: SEO 與無障礙優化 (SEO & A11y Enhancement)

## 上下文
SPA (Single Page Application) 常因內容動態載入而面臨 SEO 挑戰，且容易忽略無障礙導航體驗。為了提升應用的可搜尋性與友善度，需要引入動態 Meta 管理與無障礙功能。

## 決策
1.  **動態 Meta 管理 (MetaService)**：
    *   實作 `MetaService`，負責管理 `document.title` 與 `<meta name="description">`。
    *   整合 i18n，支援多語言標題與描述。
    *   在 `x-route` 中加入 `meta-title` 與 `meta-desc` 屬性，當路由匹配時自動更新 Meta 資訊。
2.  **跳過導航連結 (Skip Link)**：
    *   在頁面頂部加入隱藏的 "Skip to Content" 連結。
    *   當鍵盤使用者按 Tab 時顯示，允許直接跳轉至主要內容區 (`#main-content`)，提升無障礙體驗。

## 後果
- **優點**: 改善搜尋引擎索引效果，每個頁面都有獨特的標題與描述。
- **優點**: 符合 WCAG 無障礙標準，提升鍵盤操作者的使用體驗。
- **缺點**: 由於是客戶端渲染 (CSR)，對於不支援 JS 的爬蟲仍有限制（但現代爬蟲多已支援 JS）。
