# ADR 0076: 建立 Docs 與 Lab 雙向快參照

## 狀態

Accepted

## 背景

使用者明確提出教學文件與實驗室需要可快速互跳，以降低查找成本並提升手機場景下的閱讀與操作效率。

## 決策

導入雙向導覽機制，覆蓋 Docs 與 Lab 首頁兩端：

1. 在 Docs 頁面，針對可對應實驗主題提供「對應實驗室」快捷按鈕。
2. 在 Lab 首頁卡片，提供「技術手冊」快捷入口。
3. 支援 `#/docs/<doc-id>` 深連結，讓分享與跳轉更直接。
4. 保持零依賴與既有 hash routing 相容，不引入新框架。

## 後果

- **優點**：文件與實作可快速互證，教學流程更連續。
- **優點**：手機端可減少多層返回與搜尋成本。
- **缺點**：需要維護 doc-id 與 lab 路由映射表，新增主題時需同步更新。
