# ADR 0094: 原生 CSS 變數主題引擎與佈局教學實作

## 狀態
已接受 (Accepted)

## 背景
專案已進入「教育與架構」階段。雖然 `theme-service.js` 提供了基礎的主題切換，但缺乏一個直觀的實驗室頁面來展示「如何不依賴 Tailwind/Bootstrap，純靠 CSS 變數實現響應式與主題化」。

## 決策
建立一個專屬的佈局與主題實驗室頁面。
1. **動態變數注入**：實作一個介面，讓使用者即時修改 `--primary-color` 等 CSS 變數並觀察 UI 變化。
2. **響應式斷點視覺化**：展示如何利用 `clamp()` 與 `aspect-ratio` 實作自動適配佈局。
3. **三位一體同步**：建立 `LayoutPage.js`，撰寫 `layout-guide.md` 實踐篇，並更新導覽。

## 後果
- **優點**：強化專案的教育深度，讓開發者掌握現代 CSS 佈局的核心能力。
- **缺點**：CSS 變數的動態注入需小心樣式污染，應限制在實驗室容器內。
